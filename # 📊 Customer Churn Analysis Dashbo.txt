# 📊 Customer Churn Analysis Dashboard

## 📌 Overview
This project is an end-to-end Customer Churn Analysis solution built with Power BI. It includes data cleaning, SQL data preparation, Power Query transformations, DAX calculations, interactive dashboards, and a Python-based Random Forest model to predict customer churn.

## 🚀 Features
- Customer Churn Analysis Dashboard
- Churn Prediction using Random Forest
- Interactive KPIs and Visualizations
- Drill Down & Drill Through
- Dynamic Filters & Slicers
- Star Schema Data Model

## 🛠 Technologies Used
- Power BI
- SQL Server
- Power Query
- DAX
- Python (Pandas, NumPy, Scikit-learn)
- Excel

## 📈 Key KPIs
- Total Customers
- Churn Customers
- Churn Rate
- Revenue
- Monthly Charges
- Customer Segmentation
